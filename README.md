# cyber-arpg

A cyberpunk ARPG combat sandbox built in Godot 4 .NET and C#.

The project is a focused portfolio and learning piece for systems design: event-driven combat, data-driven abilities, modifiers, status effects, loot, and progression. It is intentionally scoped as a one-room combat sandbox rather than a full ARPG.

## Current Milestone

Milestone 0/1 is the technical baseline:

- One placeholder arena scene
- Player placeholder
- WASD movement
- Mouse aim direction
- Camera follow
- Placeholder Health and Energy UI

## Run

Open `project.godot` in Godot 4 .NET, build the C# project if prompted, then run the main scene.
